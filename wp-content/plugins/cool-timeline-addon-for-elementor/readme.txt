== Description ==
Use Cool Timeline WordPress plugin easily inside Elementor Page Builder and create vertical responsive history timeline.

Easily create a beautiful history or events timeline inside your WordPress website using this simple to use cool timeline addon for Elementor page builder. You need to install Cool Timeline official plugin along with this addon if you want to add timeline inside Elementor page builder.
